# 股票信息查询系统 - 升级版完整需求文档

## 项目概述

开发一个基于Web的综合投资分析平台，提供实时股票价格查询、深度基本面分析、行业分析、公司竞争力评估、投资价值分析和智能内容生成等功能。

## 核心功能需求

### 1. 股票搜索功能 ✅
- **描述**: 用户可以通过股票代码或公司名称搜索股票
- **需求详情**:
  - 支持模糊搜索（至少2个字符开始搜索）
  - 搜索结果实时显示（300ms防抖）
  - 显示股票代码和公司名称
  - 点击搜索结果后显示详细信息

### 2. 股票基本信息显示 🔴 需要升级
- **描述**: 显示股票的基本信息卡片和深度财务数据
- **需求详情**:
  - 股票代码和公司名称
  - 当前价格
  - **新增**: 最近5年关键财报数据（营收、利润、毛利率、净利率、涨跌幅）

### 3. 行业分析功能 🆕
- **描述**: 深度分析股票所在行业
- **需求详情**:
  - 股票公司业务所属行业分类
  - 行业规模、发展速度、市场占有率
  - 政策支持分析和未来前景预测
  - 行业关键指标对比

### 4. 公司竞争力分析 🆕
- **描述**: 全面评估企业核心竞争力
- **需求详情**:
  - 企业发展历程和主营业务分析
  - 商业模式和护城河分析
  - 企业文化和管理团队评估
  - 竞争对手对比分析

### 5. 投资价值分析 🆕
- **描述**: 基于券商研报的投资价值评估
- **需求详情**:
  - 整合最新券商研究报告
  - 估值分析和投资建议
  - 风险评估和投资时机判断
  - 投资评级和目标价格预测

### 6. 自定义股票列表 ✅ 已有基础
- **描述**: 用户可以创建自定义股票关注列表
- **需求详情**:
  - 支持多个分类列表（自选股、观察列表等）
  - 列表增删改查功能
  - 列表内股票排序
  - 一键刷新所有股票数据
  - 列表数据导出（Excel/CSV）

### 7. 文章生成和分享 🆕
- **描述**: 基于分析结果生成投资文章并多平台发布
- **需求详情**:
  - 一键生成股票分析文章
  - 支持多种文章格式（小红书、微信公众号、微博等）
  - 文章模板库和自定义样式
  - 社交媒体直接发布功能

### 8. 行业和股票涨跌幅排行 🆕
- **描述**: 动态显示市场热点和资金流向
- **需求详情**:
  - 今年/本月涨跌幅最大的10个行业排行
  - 今年/本月涨跌幅最大的50个企业排行
  - 实时更新和趋势分析
  - 历史数据对比功能

### 9. 响应式设计 ✅
- **描述**: 适配不同设备屏幕
- **需求详情**:
  - 桌面端：完整功能，多列布局
  - 平板端：中等布局，功能完整
  - 手机端：单列布局，简化界面
  - 加载状态和错误处理

### 10. 数据缓存机制 ✅
- **描述**: 优化API调用，提高响应速度
- **需求详情**:
  - 股票基本信息缓存5分钟
  - 历史数据缓存15分钟
  - 搜索结果缓存30分钟
  - 本地存储用户自定义列表
  - 自动过期机制

## 技术需求

### 前端技术栈
- **框架**: Next.js 14 (React 18)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图表库**: Recharts + ECharts（复杂图表）
- **图标**: Lucide React
- **HTTP客户端**: Axios
- **富文本编辑器**: TipTap
- **图表导出**: html2canvas + jsPDF

### 后端技术栈
- **框架**: Python Flask
- **数据源**: 
  - Yahoo Finance API (yfinance) ， akshare- 基础数据
  - Alpha Vantage , akshare - 财报数据
  - 金融数据API - 行业数据
  - 爬虫系统 - 券商研报
- **数据库**: SQLite + Redis (缓存)
- **CORS**: Flask-CORS
- **环境配置**: python-dotenv
- **文档生成**: python-docx
- **AI集成**: OpenAI API (文章生成)

### API接口设计（扩展）

#### 基础API ✅
- `GET /api/health` - 健康检查
- `GET /api/stock/<symbol>` - 获取股票基本信息
- `GET /api/stock/<symbol>/history` - 获取股票历史数据
- `GET /api/search` - 搜索股票
- `GET /api/popular` - 获取热门股票

#### 新增API

**财报数据API**
- `GET /api/stock/<symbol>/financials` - 获取财务数据
- `GET /api/stock/<symbol>/financials/<year>` - 获取特定年份财务数据

**行业分析API**
- `GET /api/industry/<industry_code>` - 获取行业信息
- `GET /api/industry/rankings` - 获取行业排行
- `GET /api/industry/<industry_code>/stocks` - 获取行业内股票

**公司分析API**
- `GET /api/stock/<symbol>/analysis` - 获取公司分析
- `GET /api/stock/<symbol>/competitors` - 获取竞争对手

**投资研报API**
- `GET /api/stock/<symbol>/research` - 获取投资研报
- `GET /api/stock/<symbol>/ratings` - 获取投资评级汇总

**排行榜API**
- `GET /api/rankings/sectors` - 行业涨跌幅排行
- `GET /api/rankings/stocks` - 股票涨跌幅排行

**文章生成API**
- `POST /api/articles/generate` - 生成分析文章
- `GET /api/articles/<id>` - 获取文章
- `POST /api/articles/publish` - 发布文章

**自选股管理API**（扩展）
- `POST /api/watchlist` - 创建自选股列表
- `GET /api/watchlist/<id>` - 获取自选股列表
- `PUT /api/watchlist/<id>` - 更新自选股列表
- `DELETE /api/watchlist/<id>` - 删除自选股列表
- `POST /api/watchlist/category` - 创建分类列表
